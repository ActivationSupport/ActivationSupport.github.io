Scheduling System — Quick Setup
  • 1
    Create a new Google Sheet named "Activation Scheduler"Extensions → Apps Script to open the editor.
  • 2
    Paste the 6 script filesCode.gs + booking.html + admin.html + setup.html + cancel.html + reschedule.html — file names must match exactly.
  • 3
    Run initializeSystem()Creates the Appointments, Activators, Blocked Times, and Admins sheets.
  • 4
    Deploy as Web AppDeploy → New Deployment → Web App → Execute as: Me → Access: Anyone.
  • 5
    Save Web App URL in Script PropertiesProject Settings → Script Properties → key: WEB_APP_URL
  • 6
    Run setupReminderTrigger()Enables 24-hour and 1-hour automated reminder emails.
  • 7
    Connect each Call List via SchedulerBridge.gsPaste SchedulerBridge.gs into the office sheet's Apps Script. Set SCHEDULER_SPREADSHEET_ID and SCHEDULER_WEB_APP_URL.
Default admin password is admin123 — change it in Script Properties (ADMIN_PASSWORD) before sharing the admin link.
